Demostración de habilidades.

Sitio Web - Epic Grooveshark Assembly

Sitio web para la comunidad Epic Grooveshark Assembly (EGSA), quienes cuentan con un broadcast en Grooveshark.

Juego Javascript: Space Game

Juego simple hecho con Javascript en el que se demuestra la detección de colisión basada en la información de los pixeles dibujados en la pantalla.

Juego Javascript: Cuccos for Dinner

Juego del tipo shooter hecho en Javascript utilizando la librería CreateJS. En este se demuestran el manejo de animaciones(EaselJS) y sonidos(SoundJS), además de la implementación de una cola para cargar los recursos necesarios(PreloadJS).

Juego Javascript: Platformer

Primer acercamiento al framework Phaser, el cual facilita el proceso de desarrollo de videojuegos en HTML5.
